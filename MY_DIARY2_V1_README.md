# マイ日記帳2（バージョン1）

## 📝 概要
**マイ日記帳2**は、マイ日記帳1（v5.2）と全く同じ機能を持つ独立した日記アプリです。

---

## ✨ 特徴

### 完全独立動作
- **データ分離**: マイ日記帳1とデータが完全に分離
- **ローカルストレージキー**:
  - `diary2_articles`（記事データ）
  - `diary2_profile`（プロフィールデータ）
  - `diary2_custom_links`（カスタムリンク設定）

### 同じ機能
1. **ダッシュボード**
   - 年タブ（2026/2025/2024...）
   - 12ヶ月ボックス（6列×2行）
   - 各月1-5日表示、もっと見るで全日付展開

2. **記事機能**
   - ブロックエディタ（テキスト/画像を自由配置）
   - 下書き保存機能
   - 複数画像アップロード
   - タグ・カテゴリー
   - コメント機能

3. **プロフィール**
   - 写真、名前、誕生日、血液型、星座、身長
   - カスタム項目（複数追加可能）

4. **カスタムリンク**
   - ヘッダーに最大2つのリンクボタン
   - タイトルとURLを編集可能

5. **検索・エクスポート/インポート**
   - 全文検索
   - JSONエクスポート/インポート

---

## 🚀 GitHub公開方法

### ステップ1: リポジトリ作成
1. GitHubで新しいリポジトリ作成
2. リポジトリ名: `my-diary2`（または任意の名前）
3. Public を選択
4. README.md を追加（オプション）

### ステップ2: ファイルアップロード
1. `my-diary2.html` を `index.html` にリネーム
2. GitHubリポジトリにアップロード

### ステップ3: GitHub Pages有効化
1. Settings → Pages
2. Source: `main` branch
3. Root を選択
4. Save

### ステップ4: アクセス
数分後、以下のURLでアクセス可能：
```
https://あなたのユーザー名.github.io/my-diary2/
```

## 🔗 マイ日記帳1とのリンク設定

両方のサイトでカスタムリンクを設定すると、相互にリンクできます：

### マイ日記帳1の設定
1. ✏️ボタンをクリック
2. タイトル: `マイ日記帳2`

4. 保存

### マイ日記帳2の設定
1. ✏️ボタンをクリック
2. タイトル: `マイ日記帳1`
3. URL: `https://あなたのユーザー名.github.io/my-diary/`
4. 保存

## 💾 データ管理

### エクスポート
1. 「エクスポート」ボタンをクリック
2. `diary_backup_YYYY-MM-DD.json` がダウンロードされる
3. Google Drive/Dropbox等に保存

### インポート
1. 「インポート」ボタンをクリック
2. バックアップJSONファイルを選択
3. データが復元される

### 注意事項
- マイ日記帳1とマイ日記帳2のデータは完全に独立
- バックアップファイルも別々に管理してください

## 🆘 サポート

### よくある質問

**Q: マイ日記帳1とデータは共有されますか？**
A: いいえ、完全に独立しています。

**Q: 両方を同時に開けますか？**
A: はい、別のブラウザタブで開けます。

**Q: データが消えることはありますか？**
A: ローカルストレージを使用しているため、ブラウザのキャッシュクリアで消える可能性があります。定期的にエクスポートしてください。

**Q: スマホでも使えますか？**
A: はい、レスポンシブデザインでスマホでも使えます。
